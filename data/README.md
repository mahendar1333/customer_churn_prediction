# Dataset Information

This project uses the Telco Customer Churn dataset from Kaggle.

## How to get the data:
1. Go to: https://www.kaggle.com/datasets/blastchar/telco-customer-churn
2. Download the dataset
3. Place `WA_Fn-UseC_-Telco-Customer-Churn.csv` in this folder

## File Size
The CSV file is approximately 4MB and is excluded from Git via .gitignore
